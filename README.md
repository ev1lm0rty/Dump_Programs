# About

This script dumps domains that are up for bug bounty from
* Bugcrowd
* Hackerone
* Intigriti
* YesWeHack
* Other


# Usage
* `./dump_programs.sh` to dump all publicly open bug bounty domains.
* `./dump_programs.sh s` to dump domains and their subdomains (Takes time).


# Prequisites
* Install subfinder
* Install curl
